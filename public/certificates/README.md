# Certificates Folder

Place your certificate PDF files here with these exact names:

1. `data-scientist-workshop.pdf` - Train to Transform – Data Scientist Workshop
2. `joy-of-computing-python.pdf` - The Joy of Computing Using Python – NPTEL
3. `hpc-nptel.pdf` - Practical High-Performance Computing – NPTEL
4. `cloud-computing-simplilearn.pdf` - Introduction to Cloud Computing – Simplilearn
5. `diploma-computer-application.pdf` - Diploma in Computer Application

## File Naming Convention

- Use lowercase letters
- Replace spaces with hyphens (-)
- Use `.pdf` extension
- Keep names descriptive but concise

## Example

```
public/
└── certificates/
    ├── data-scientist-workshop.pdf
    ├── joy-of-computing-python.pdf
    ├── hpc-nptel.pdf
    ├── cloud-computing-simplilearn.pdf
    └── diploma-computer-application.pdf
```

Once you add the PDF files, the certificate links will work automatically!
